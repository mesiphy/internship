## 定义
拒答策略 是 Prompt 与 Structured Output 中的关键知识点，用于理解和组织 模型指令设计、输出约束与交互稳定性。

## 核心问题
核心问题是：拒答策略 在 Prompt 与 Structured Output 里承担什么作用、与哪些上下游环节相连、如何判断它是否发挥作用。

## 应用场景
- 为模型定义任务边界、角色和输出格式。
- 让模型输出稳定的 JSON 或结构化结果。
- 提升单轮交互、工具调用和表单生成的稳定性。

## 常见误区
- 只追求提示词花样，不约束输入输出边界。
- 依赖自然语言输出而不做结构化校验。
