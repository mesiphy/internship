## 定义
Prompt 组合 是 Prompt 与 Structured Output 中的一项核心能力或机制，用于组织和发挥 模型指令设计、输出约束与交互稳定性。

## 核心问题
核心问题是：Prompt 组合 能解决什么问题、边界在哪里、应该和哪些机制组合使用。

## 应用场景
- 为模型定义任务边界、角色和输出格式。
- 让模型输出稳定的 JSON 或结构化结果。
- 提升单轮交互、工具调用和表单生成的稳定性。

## 常见误区
- 只追求提示词花样，不约束输入输出边界。
- 依赖自然语言输出而不做结构化校验。
