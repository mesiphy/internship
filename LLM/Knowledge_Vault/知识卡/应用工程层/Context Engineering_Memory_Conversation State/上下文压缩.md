## 定义
上下文压缩 是 Context Engineering / Memory / Conversation State 中常见的工程处理策略，用来稳定 上下文组织、记忆管理与多轮状态控制 对应的链路表现。

## 核心问题
核心问题是：什么时候需要用 上下文压缩、会带来什么代价、该如何与上下游策略配合。

## 应用场景
- 管理多轮对话中的历史、状态和长期记忆。
- 控制长上下文、摘要压缩和检索注入的平衡。
- 为 Agent 设计隔离上下文和状态传递方式。

## 常见误区
- 把所有信息都塞进上下文而不做选择和压缩。
- 把记忆、会话状态和任务状态混为一谈。
