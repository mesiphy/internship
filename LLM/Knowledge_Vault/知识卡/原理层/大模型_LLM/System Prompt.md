## 定义
System Prompt 是 大模型 / LLM 中的一项核心能力或机制，用于组织和发挥 生成式模型的输入、输出、能力控制与成本权衡。

## 核心问题
核心问题是：System Prompt 能解决什么问题、边界在哪里、应该和哪些机制组合使用。

## 应用场景
- 设计基于 LLM 的问答、生成和助手类功能。
- 控制输出格式、成本、延迟和推理能力。
- 判断该靠 prompt、架构还是模型切换来优化效果。

## 常见误区
- 把 prompt 调整当成所有问题的万能解。
- 忽略上下文、工具、成本和延迟之间的联动。
